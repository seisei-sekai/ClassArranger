# 教程系统使用说明

## 功能概述

教程系统是一个交互式的学习工具，帮助老师快速了解排课系统的核心功能。通过四个示例课程，老师可以学习如何：
- 查看课程详情
- 拖拽调整时间
- 双击编辑名称
- 使用AI自动排课

---

## 核心特性

### 1. **一键切换**
- 点击"显示教程"按钮，立即加载4个示例课程
- 再次点击"隐藏教程"按钮，清除所有示例数据
- 切换不影响用户自己创建的课程

### 2. **分步指导**
- **步骤一：查看课程详情** - 学习如何点击课程查看信息
- **步骤二：拖拽调整时间** - 学习如何拖动课程到新时间
- **步骤三：双击快速编辑** - 学习如何快速修改课程名称
- **步骤四：AI自动排课** - 学习如何使用智能排课功能

### 3. **视觉反馈**
- 教程按钮有明显的激活状态（深蓝色背景）
- 教程面板采用醒目的渐变色头部
- 每个步骤都有清晰的编号和说明
- 实操示例用灰色背景框标注

### 4. **响应式设计**
- 桌面端：步骤横向排列，信息完整
- 移动端：步骤纵向堆叠，自适应布局
- 所有设备都有完美的阅读体验

---

## 使用方法

### 第一步：启动教程
1. 访问排课页面：`http://localhost:3000/XdfClassArranger_function`
2. 点击右上角的"显示教程"按钮（书本图标）
3. 教程面板自动展开，日历上出现4个示例课程

### 第二步：跟随指导
1. **查看详情**：点击"步骤一：面试练习 - 张三"
   - 弹出的模态框显示学生、教师、教室等信息
   
2. **拖拽课程**：按住"步骤二：志望理由书 - 王五"
   - 拖到其他日期或时间
   - 松开鼠标完成调整

3. **快速编辑**：双击"步骤三：EJU日语 - 李四"
   - 输入新名称
   - 按回车确认或ESC取消

4. **AI排课**：点击"AI自动排课"按钮
   - 观察进化过程
   - 查看冲突分析

### 第三步：清除教程
1. 点击"隐藏教程"按钮
2. 所有示例课程自动清除
3. 用户自己创建的课程保留不变

## [技术] 技术实现

### 核心代码结构

#### 1. 状态管理
```javascript
const [showTutorial, setShowTutorial] = useState(false);
```

#### 2. 教程数据
```javascript
const tutorialEvents = [
  {
    id: 'tutorial-1',
    title: '步骤一：面试练习 - 张三',
    start: '2025-12-02T10:00:00',
    end: '2025-12-02T12:00:00',
    // ... 更多属性
  },
  // ... 其他3个课程
];
```

#### 3. 切换函数
```javascript
const handleToggleTutorial = () => {
  if (showTutorial) {
    // 清除教程数据
    setEvents(events.filter(e => !e.id.startsWith('tutorial-')));
    setShowTutorial(false);
  } else {
    // 加载教程数据
    const nonTutorialEvents = events.filter(e => !e.id.startsWith('tutorial-'));
    setEvents([...nonTutorialEvents, ...tutorialEvents]);
    setShowTutorial(true);
  }
};
```

### 数据过滤逻辑
- 通过 `id.startsWith('tutorial-')` 识别教程数据
- 切换时只影响教程数据，保留用户数据
- 使用数组过滤和扩展运算符实现

## [工具] 自定义教程

### 修改教程内容
编辑 `Function.js` 中的 `tutorialEvents` 数组：

```javascript
const tutorialEvents = [
  {
    id: 'tutorial-1',  // 必须以 tutorial- 开头
    title: '你的课程标题',
    start: '2025-12-02T10:00:00',  // 开始时间
    end: '2025-12-02T12:00:00',    // 结束时间
    backgroundColor: '#5A6C7D',    // 背景颜色
    extendedProps: {
      student: '学生姓名',
      teacher: '教师姓名',
      campus: '校区',
      room: '教室',
      description: '课程描述'
    }
  },
  // ... 添加更多课程
];
```

### 修改步骤说明
编辑 `Function.js` 中的教程面板JSX：

```jsx
<div className="tutorial-step">
  <div className="step-number">1</div>
  <div className="step-content">
    <h4>你的步骤标题</h4>
    <p>你的步骤说明</p>
    <div className="step-example">示例：你的示例说明</div>
  </div>
</div>
```

### 调整样式
编辑 `Function.css` 中的教程相关样式：

```css
.tutorial-panel {

  border-radius: 8px;         /* 圆角大小 */
}

.tutorial-header {
  background: linear-gradient(135deg, #5A6C7D 0%, #6B7C6E 100%);
  /* 修改渐变色 */
}

.step-number {
  background: linear-gradient(135deg, #5A6C7D 0%, #6B7C6E 100%);
  /* 修改步骤编号背景 */
}
```

## [学习] 教学建议

### 给老师的提示
1. **首次使用**：建议先启动教程模式，跟着步骤操作一遍
2. **反复练习**：可以多次切换教程，熟悉每个功能
3. **逐步深入**：先掌握基础功能（查看、拖拽），再尝试高级功能（AI排课）
4. **实际应用**：熟悉后关闭教程，开始创建实际课表

### 常见问题解答

**Q: 教程数据会影响我的实际课表吗？**
A: 不会。教程数据是临时的，点击"隐藏教程"即可清除。

**Q: 我可以修改教程中的示例课程吗？**
A: 可以。教程课程和普通课程一样可以拖拽和编辑。

**Q: 教程数据会被保存吗？**
A: 不会。刷新页面后教程数据会消失，需要重新点击"显示教程"。

**Q: 可以增加更多教程步骤吗？**
A: 可以。按照"自定义教程"部分的说明添加更多课程和步骤。
