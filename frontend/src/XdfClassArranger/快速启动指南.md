## [移动] 基本操作（5分钟上手）

### + 添加课程
1. 在日历上点击任意日期
2. 输入课程名称
3. 完成！

### [查看]️ 查看/删除课程
1. 点击日历上的课程
2. 查看详细信息
3. 点击"确定"可删除

### ⟲ 修改时间
- **拖动课程** = 改变开始时间
- **拖动课程边缘** = 调整时长

### [日历] 切换视图
点击右上角按钮：
- **月** = 月视图（整月概览）
- **周** = 周视图（带时间轴）
- **日** = 日视图（单日详情）
- **列表** = 列表视图（简洁明了）

## [!] 注意事项

1. **时间格式**: 使用 ISO 8601 格式（`2025-12-10T10:00:00`）
2. **事件ID**: 必须唯一
3. **颜色**: 使用十六进制（`#4CAF50`）
4. **时区**: 已设置为 `Asia/Tokyo`

## 🆘 遇到问题？

### 常见问题
1. **事件不显示** → 检查 `start` 时间格式
2. **无法拖动** → 确认 `editable={true}`
3. **中文显示错误** → 检查 `locale="zh-cn"`

## [特性] 代码示例

### 添加一个新课程
```javascript
const newEvent = {
  id: String(Date.now()),
  title: '1v1托福 - 学生名',
  start: '2025-12-20T14:00:00',
  end: '2025-12-20T16:00:00',
  backgroundColor: '#E91E63',
  extendedProps: {
    student: '学生名',
    teacher: '教师名',
    campus: '旗舰校',
    room: '个别指导室5'
  }
};
setEvents([...events, newEvent]);
```

### 修改配置
```javascript
// 改成从早上6点到晚上11点
slotMinTime="06:00:00"
slotMaxTime="23:00:00"

// 改成15分钟间隔
slotDuration="00:15:00"

// 隐藏周末
weekends={false}
```
